**Based on the official 25-page "DevOps Learning Resources Syllabus" (April 2026)**  
Companion to the 28-Day DevOps Infrastructure Crash Course

Philosophy, Technical Depth, and Hands-On Practice

## How to Use This Document

This syllabus is designed to complement your 28-day course. Each section builds on the last — philosophy gives context to technical skills, and hands-on practice makes the philosophy concrete.  

Start with **Section 7: Recommended Learning Path** to see the phased timeline. Use the other sections as reference. Resources marked **FREE** cost nothing. Resources marked **ESSENTIAL** are highest priority.

## Section 1: The DevOps Philosophy — Essential Reading

These books explain **WHY** DevOps exists. Read them early for context.

**Must Read (Start Here)**
- **[The Phoenix Project](https://itrevolution.com/product/the-phoenix-project)** by Gene Kim, Kevin Behr, George Spafford (Novel)  
  The single most influential DevOps book. Read this first.

**Essential**
- **[The DevOps Handbook (2nd Edition)](https://itrevolution.com/product/the-devops-handbook-second-edition)** by Gene Kim, Jez Humble, et al.
- **[Accelerate](https://itrevolution.com/product/accelerate)** by Nicole Forsgren, Jez Humble, Gene Kim
- **[The Unicorn Project](https://itrevolution.com/product/the-unicorn-project)** by Gene Kim (developer perspective)

**Google SRE Series (Free Online)**
- [Site Reliability Engineering](https://sre.google/sre-book/table-of-contents/)
- [The SRE Workbook](https://sre.google/workbook/table-of-contents/)
- [Building Secure & Reliable Systems](https://sre.google/books/)

## Section 7: Recommended Learning Path

**Phase 1 — Foundation (Week 1)**  
Phoenix Project + Linux Command Line + Nana intro videos

**Phase 2 — Deepening (Weeks 2–3)**  
DevOps Handbook + Terraform/Ansible courses + hands-on labs

**Phase 3 — Broadening (Weeks 4–8)**  
Accelerate, SRE books, KodeKloud labs, GitHub portfolio building

**Phase 4 — Interview Prep (Weeks 9–12)**  
Unicorn Project + mock interviews + podcasts
